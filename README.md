# AI Chat Navigation Bar (对话导航栏)

这是一个为 ChatGPT 和 Google Gemini 网页版设计的浏览器插件。它在页面右侧提供一个浮动的消息导航栏，帮助你快速定位、管理长对话，支持收藏重点内容并导出。

## ✨ 主要功能

- **🚀 快速导航**: 自动提取对话流中的提问和回答，点击右侧目录即可平滑滚动到对应位置。
- **🤖 多平台适配**: 
  - ✅ **ChatGPT** (chatgpt.com / chat.openai.com)
  - ✅ **Google Gemini** (gemini.google.com)
- **⭐ 重点收藏**: 点击星星收藏特定消息，支持“仅显示收藏”筛选模式，快速回顾关键点。
- **📝 以此为书签**: 双击导航条目或点击编辑图标，可自定义该条消息的标题（例如：“项目需求”、“代码修正版”），方便查找。
- **💾 导出 Markdown**: 支持将“已收藏”的消息一键导出为 `.md` 文件，方便保存到笔记软件（如 Obsidian, Notion）。
- **🎨 极简设计**: 半透明悬浮窗设计，支持折叠，不打扰正常阅读。

## 🛠️ 安装方法 (开发者模式)

1. 下载本项目代码（Download ZIP）并解压。
2. 打开 Chrome 或 Edge 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 打开右上角的 **开发者模式 (Developer mode)** 开关。
4. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择解压后的插件文件夹（包含 `manifest.json` 的那个目录）。
6. 打开 ChatGPT 或 Gemini 页面刷新即可看到效果。

## 📖 使用指南

- **跳转**: 点击列表项。
- **折叠**: 点击面板右上角的 `≡` 按钮可收起面板。
- **收藏**: 点击条目上的 `☆`。
- **筛选**: 点击顶部栏的 `★` 按钮，只看收藏项。
- **重命名**: 双击文字或点击 `✎` 图标，输入新名字后回车。
- **导出**: 点击顶部栏的 `⤓` 按钮导出已收藏内容。

## 🔒 隐私声明

本插件纯本地运行，所有数据（收藏状态、重命名备注）均存储在您的浏览器本地 (`chrome.storage.local`)，不会上传到任何外部服务器。
